# Alpha QTI API

# Overview

[1EdTech QTI](https://www.1edtech.org/standards/qti/index) (Question and Test Interoperability) is an open standard created by the 1EdTech Consortium developed to ensure consistent creation, delivery, and reporting of digital assessments. It enables interoperability between different assessment tools and learning management systems, making it easier to share, reuse, and analyze test items and assessment results across various platforms.
